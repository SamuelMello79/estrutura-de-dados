# estrutura-dados

Um repositório para armazenar os desafios propostos em aula na matéria de Estrutura de Dados.

Todos os desafios serão resolvidos em C, com a resolução inicial, e depois uma versão aprimorada.

Conforme o respositório for aumentando é capaz que eu altere a organização das pastas, mas por enquanto será organizado por atividade.

Espero poder ajudar alguém com esses pequenos exemplos.
